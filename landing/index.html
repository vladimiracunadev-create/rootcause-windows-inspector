Funciones

Todo lo que necesitas para diagnosticar

Sin ruido. Sin magia. Solo datos concretos sobre lo que está pasando.

📊

Semáforo y sparklines

Estado 🟢🟡🔴 en tiempo real con mini-gráficos de tendencia para CPU, RAM e I/O de las últimas 60 muestras.

⚙️

Procesos dominantes

Tabla con severidad, puntaje, I/O delta, command line completo y filtros por crítico / aviso / normal.

🌐

Conexiones de red

IPs públicas activas por proceso con bloqueo directo vía firewall de Windows desde la UI o CLI.

🗂️

Temporales y cachés

Escaneo de %TEMP%, SoftwareDistribution, DeliveryOptimization. Alertas cuando crecen anormalmente.

📜

Historial SQLite + A vs B

Últimas 60 capturas persistidas. Compara dos momentos con deltas de CPU / RAM / I/O / Alertas en verde o rojo.

🔬

Modo de precisión ETW/WPR

Captura ETL con WPR desde la UI. Resume automáticamente con tracerpt y heurísticas propias sin salir de la app.

💻

CLI completa

rootcause status, snapshot, history, export, kill, block-ip, wpr — todo desde la consola.

⌨️

Atajos de teclado

F5 actualizar · Ctrl+E exportar · Ctrl+1…8 navegar tabs sin soltar el teclado.

🖥️

Hardware del equipo

OS, CPU, núcleos, frecuencia y RAM total visibles en el tab Resumen y en el tab Acerca del producto.

Cómo funciona

Dos capas de diagnóstico

1

Modo operativo liviano

El modo principal. Bajo consumo. Útil para observación frecuente sin añadir carga al sistema.

  • ▸ Semáforo general del estado del equipo
  • ▸ Sparklines de CPU · RAM · I/O en tiempo real
  • ▸ Top de procesos con severidad y puntaje
  • ▸ Escaneo de cachés y temporales de Windows
  • ▸ Conexiones activas con correlación a IP
  • ▸ Historial SQLite + comparación A vs B
  • ▸ Exportación JSON de evidencia
2

Modo de precisión ETW/WPR

Para cuando la observación liviana no basta. Captura y resume trazas ETL del sistema sin salir de la app.

  • ▸ Iniciar / detener / cancelar captura WPR
  • ▸ Registrar contexto del problema
  • ▸ Resumir último ETL con tracerpt
  • ▸ Barra visual de proveedores ETW activos
  • ▸ Exportar trace-analysis.json
  • ▸ Ruta clara a WPA para análisis profundo

CLI

Funciona también desde la consola

El mismo binario que la GUI. Sin instalación adicional.

Diagnóstico 
# Estado rápido
rootcause status

# Snapshot completo a JSON
rootcause snapshot

# Últimas 20 capturas del historial
rootcause history 20

# Exportar evidencia
rootcause export
Intervención + WPR 
# Finalizar proceso
rootcause kill 4892

# Bloquear IP vía firewall
rootcause block-ip 185.220.101.14

# Captura ETL
rootcause wpr start --note "disco al 100%"
rootcause wpr stop
rootcause wpr analyze

Requisitos

Qué necesitas para correrlo

⚙️ Sistema mínimo

  • ✓ Windows 10 x64 o Windows 11 x64
  • ✓ 2 núcleos de CPU
  • ✓ 8 GB RAM
  • ✓ 2 GB libres en disco (SSD recomendado)
  • ✓ 1366×768 como mínimo funcional
  • △ Administrador para bloquear IP o detener servicios

🚀 Recomendado

  • ✦ Windows 11 x64 actualizado
  • ✦ 4+ núcleos de CPU
  • ✦ 16 GB RAM o más
  • ✦ SSD NVMe · 10 GB libres
  • ✦ 1920×1080 o mayor
  • ✦ WPT instalado para modo precisión ETW

🔨 Para compilar desde fuente

Obligatorio

  • • Rust estable (rustup)
  • • Toolchain MSVC
  • • Visual Studio Build Tools

Recomendado

  • • PowerShell 5+
  • • Git
  • • cargo fmt / clippy

Para empaquetado

  • • Inno Setup (iscc.exe)
  • • wpr.exe (modo ETW)
  • • tracerpt.exe (resumen ETL)

Instalación

Tres formas de empezar

Más fácil

Instalador Inno Setup

  1. 1. Descarga RootCause-Setup.exe desde la sección de Descargas
  2. 2. Ejecuta el instalador (o modo silencioso con /VERYSILENT /SUPPRESSMSGBOXES /NORESTART)
  3. 3. Busca RootCause en el menú Inicio
Portable

ZIP portable

# Extrae el ZIP y ejecuta directamente
.\rootcause.exe

# O desde consola
.\rootcause.exe --help
Desde fuente

Compilar con Cargo

# Verificar entorno
.\scripts\verify-environment.ps1

# Compilar release
cargo build --release

# Ejecutar (algunos features requieren administrador)
.\target\release\rootcause.exe

# Empaquetar como instalador
.\scripts\package-inno.ps1
